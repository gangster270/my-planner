# 🌱 주간 업무 플래너

브라우저에서 바로 실행할 수 있는 **단일 HTML 기반 주간·월간 업무 플래너**입니다.  
업무/개인 할 일을 날짜별로 관리하고, 우선순위, 기한, D-day, 완료율, 업무 기록, 위젯 모드, 데이터 내보내기/불러오기 기능을 제공합니다.

---

## 1. 프로젝트 개요

이 프로젝트는 별도의 서버나 데이터베이스 없이 `index.html` 파일 하나로 실행되는 개인 업무 관리 도구입니다.

주요 목적은 다음과 같습니다.

- 주간 단위로 해야 할 일을 직관적으로 관리
- 업무와 개인 일정을 구분하여 관리
- 우선순위와 기한을 설정하여 마감 관리
- 완료율과 남은 업무를 한눈에 확인
- 하루별 업무 기록 작성
- JSON 파일로 데이터 백업 및 복원
- 작은 화면에서 볼 수 있는 위젯 모드 제공

---

## 2. 주요 기능

### 2.1 주간 보기

- 월요일부터 일요일까지 7일 단위로 할 일을 표시합니다.
- 각 날짜별로 할 일을 추가할 수 있습니다.
- 오늘 날짜는 강조 표시됩니다.
- 완료한 일은 체크하여 완료 처리할 수 있습니다.
- 미완료 할 일을 다음 주 같은 요일로 이월할 수 있습니다.

### 2.2 월간 보기

- 한 달 전체 일정을 달력 형태로 확인할 수 있습니다.
- 날짜를 클릭하면 해당 날짜의 상세 화면을 열 수 있습니다.
- 날짜별 할 일 일부가 미리보기로 표시됩니다.

### 2.3 할 일 관리

각 할 일에는 다음 정보를 설정할 수 있습니다.

| 항목 | 설명 |
|---|---|
| 내용 | 할 일 제목 또는 메모 |
| 분류 | 업무 / 개인 |
| 우선순위 | 높음 / 보통 / 낮음 |
| 기한 | D-day 계산에 사용 |
| 완료 여부 | 체크박스로 완료 처리 |

### 2.4 업무/개인 분류

- `💼 업무`
- `🏠 개인`

상단 필터를 통해 전체, 업무, 개인 할 일을 따로 볼 수 있습니다.

### 2.5 우선순위

- 🔴 높음
- 🟡 보통
- ⚪ 낮음

할 일 카드의 작은 점을 클릭하면 우선순위가 순환 변경됩니다.

### 2.6 D-day 표시

기한을 설정하면 다음과 같이 표시됩니다.

| 상태 | 표시 예시 |
|---|---|
| 기한 지남 | D+1, D+2 |
| 오늘 마감 | D-DAY |
| 남은 기한 | D-1, D-3 |

기한이 지난 미완료 업무는 별도로 강조 표시됩니다.

### 2.7 업무 기록

각 날짜별로 업무 기록을 작성할 수 있습니다.

예시:

```text
- 오전 회의 참석
- 자료 취합 완료
- 스마트팜 데이터 정리
- 다음 주 보고자료 초안 작성
```

작성한 기록은 브라우저에 자동 저장됩니다.

### 2.8 위젯 모드

`🪟 위젯` 버튼을 누르면 좁은 화면에 적합한 위젯 형태로 전환됩니다.

위젯 모드에서는 다음 범위를 선택할 수 있습니다.

- 이번 주
- 3일

### 2.9 데이터 내보내기 / 불러오기

하단 버튼을 통해 데이터를 JSON 파일로 저장하거나 다시 불러올 수 있습니다.

- `데이터 내보내기`: 현재 데이터를 `.json` 파일로 다운로드
- `불러오기`: 이전에 저장한 JSON 파일을 불러오기

## 3. 실행 방법

이 프로젝트는 HTML, CSS, JavaScript가 하나의 파일에 포함된 구조입니다.  
따라서 별도의 설치 과정 없이 브라우저에서 바로 실행할 수 있습니다.

### 방법 1. 파일을 브라우저로 열기

1. 코드를 `index.html` 파일로 저장합니다.
2. 저장한 `index.html` 파일을 더블클릭합니다.
3. 브라우저에서 주간 업무 플래너가 실행됩니다.

### 방법 2. VS Code Live Server 사용

VS Code를 사용하는 경우 다음 방식도 가능합니다.

1. 프로젝트 폴더를 VS Code로 엽니다.
2. `index.html` 파일을 생성하고 코드를 붙여넣습니다.
3. Live Server 확장 프로그램을 설치합니다.
4. `index.html`에서 마우스 우클릭 후 `Open with Live Server`를 선택합니다.

## 5. 데이터 저장 방식

이 플래너는 서버가 아니라 브라우저의 `localStorage`에 데이터를 저장합니다.

사용되는 저장 키는 다음과 같습니다.

```javascript
const KEY = 'weeklyPlanner_v1';
```

저장되는 주요 데이터 구조는 다음과 같습니다.

```json
{
  "tasks": {
    "2026-06-22": [
      {
        "id": "고유ID",
        "text": "할 일 내용",
        "priority": "mid",
        "category": "work",
        "done": false,
        "due": "2026-06-22"
      }
    ]
  },
  "logs": {
    "2026-06-22": "업무 기록 내용"
  }
}
```

## 6. 사용 방법

### 6.1 할 일 추가

주간 보기에서 원하는 날짜 아래의 입력창에 할 일을 입력한 뒤 `Enter`를 누르면 추가됩니다.

### 6.2 할 일 완료

할 일 왼쪽의 체크박스를 클릭하면 완료 처리됩니다.  
완료된 할 일은 흐리게 표시되고 취소선이 적용됩니다.

### 6.3 할 일 수정

할 일 텍스트를 클릭하면 수정 창이 열립니다.  
수정 창에서 다음 항목을 변경할 수 있습니다.

- 할 일 내용
- 업무/개인 분류
- 우선순위
- 기한
- 삭제

### 6.4 우선순위 변경

할 일 카드의 작은 색상 점을 클릭하면 우선순위가 다음 순서로 변경됩니다.

```text
높음 → 보통 → 낮음 → 높음
```

### 6.5 업무/개인 전환

할 일 카드의 `💼` 또는 `🏠` 아이콘을 클릭하면 업무/개인 분류가 전환됩니다.

### 6.6 업무 기록 작성

각 날짜 하단의 `📝 업무 기록` 버튼을 클릭하면 하루 상세 창이 열립니다.  
여기서 해당 날짜의 업무 기록을 작성할 수 있습니다.

### 6.7 미완료 업무 이월

`미완료 다음 주로 이월 →` 버튼을 누르면 현재 주의 미완료 할 일이 다음 주 같은 요일로 복사됩니다.  
완료된 할 일은 현재 주에 남고, 미완료 할 일은 다음 주로 이동됩니다.

### 6.8 데이터 백업

하단의 `데이터 내보내기` 버튼을 누르면 현재 데이터가 JSON 파일로 저장됩니다.  
브라우저 캐시 삭제, PC 변경, 브라우저 변경에 대비하여 주기적으로 백업하는 것을 권장합니다.

## 9. 외부 의존성

이 프로젝트는 대부분 순수 HTML/CSS/JavaScript로 동작합니다.  
다만 한글 폰트 표시를 위해 Pretendard 웹폰트를 CDN으로 불러옵니다.

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/orioncactus/pretendard/dist/web/static/pretendard.css">
```

인터넷 연결이 없어도 기능 자체는 동작하지만, 웹폰트는 적용되지 않을 수 있습니다.

## 10. 주의사항

### 10.1 브라우저별 데이터 분리

데이터는 현재 사용하는 브라우저의 `localStorage`에 저장됩니다.  
따라서 Chrome에서 입력한 데이터가 Edge나 다른 PC에 자동으로 공유되지는 않습니다.

### 10.2 캐시 삭제 주의

브라우저 데이터 또는 사이트 데이터를 삭제하면 저장된 할 일이 사라질 수 있습니다.  
중요한 데이터는 반드시 `데이터 내보내기` 기능으로 백업하는 것이 좋습니다.

### 10.3 협업 기능 없음

현재 버전은 개인용 플래너입니다.  
여러 사용자가 동시에 접속하거나 데이터를 공유하는 기능은 포함되어 있지 않습니다.

### 10.4 서버 저장 기능 없음

현재 데이터는 서버가 아닌 브라우저 내부에만 저장됩니다.  
장기적으로 사용하려면 Google Drive, Firebase, Supabase, Notion API 등과 연동하는 방식으로 확장할 수 있습니다.

## 12. 농업 연구 업무용 활용 예시

이 플래너는 일반 업무뿐만 아니라 농업 연구 일정 관리에도 활용할 수 있습니다.

예시 업무:

```text
- 잎들깨 생육조사
- LI-600 생리반응 측정
- 온실 환경데이터 다운로드
- 결측값 QC
- 수확량 조사
- 분석 코드 수정
- 주간 보고자료 작성
- 농가 현장 컨설팅 일정 정리
```

예시 업무 기록:

```text
2026-06-22 업무 기록

- B5_1~B5_8 구역 환경데이터 다운로드
- PPFD 센서 결측 구간 확인
- 생육조사 데이터 입력
- DLI 계산 코드 검토
- 다음 조사일정 확인
```
